# AMO Listing Screenshots

Capture and place review screenshots for AMO submission in this folder locally.

Suggested captures:

1. Popup ready state on an Overleaf project page.
2. Export completed state with CSV/XLSX options shown.
3. Example output file opened in a spreadsheet viewer.

Recommended format:

- PNG or JPG
- At least 1280x800
- Avoid personal/private document content

These image files are ignored by git via the repository `.gitignore`.
